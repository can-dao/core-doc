#  Select 选择器
###### 当选项过多时，使用下拉菜单展示并选择内容。

``` html
<select class="di-select">
  <option value ="请选择">请选择</option>
  <option value ="香菇饭">香菇饭</option>
  <option value="肥牛饭">肥牛饭</option>
  <option value="烧鹅饭">烧鹅饭</option>
</select>
```
